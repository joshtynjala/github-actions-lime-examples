# Github Actions with Lime Examples

Learn to set up [Github Actions](https://github.com/actions/) to build your [Lime](https://lime.software/) projects written in [Haxe](https://haxe.org/).

Examples created by [Josh Tynjala](https://github.com/sponsors/joshtynjala), the author of [Feathers UI](https://feathersui.com/) and a member of the [OpenFL](https://openfl.org/) leadership team.
